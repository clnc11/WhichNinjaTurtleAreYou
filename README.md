# Which Ninja Turtle Are You?

A mobile-friendly personality quiz that assigns the player Leonardo, Raphael, Donatello, or Michelangelo.

## Features

- Username entry
- 12 personality questions
- Four-result scoring system
- Responsive phone and desktop layout
- Original turtle-inspired artwork
- Downloadable PNG result card
- No frameworks or external files required

## Run locally

Double-click `index.html`, or open it in any modern browser.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save, then open the GitHub Pages URL after deployment finishes.

## Customize

All questions, results, descriptions, and colors are inside `index.html`.
